# ShenNong-LLM-docker

一个基于 Docker 的前后端智能问答系统，支持专业医疗问答和生活问答。

## 前置条件
- 安装 [Docker](https://docs.docker.com/get-docker/) 和 [Docker Compose](https://docs.docker.com/compose/install/).
- 准备模型文件：
  - （专业模型）
  - （生活模型）
- GPU 支持（NVIDIA 驱动和 CUDA）。

## 运行项目
1. **下载 docker-compose.yml**
   - 下载 `docker-compose.yml` 。

   - 或手动创建 docker-compose.yml文件
   
     - 直接从dockerhub下载  可能会有网络问题
   
     ```python
     services:
       backend:
         image: jasonc25/backend:latest
         ports:
           - "8000:8000"
         volumes:
           - "${PROFESSIONAL_MODEL_DIR}:/models/deepseek-r1-7B"
           - "${LIVING_MODEL_DIR}:/models/qwen-chat-7b"
         environment:
           - PROFESSIONAL_MODEL_PATH=/models/deepseek-r1-7B
           - LIVING_MODEL_PATH=/models/qwen-chat-7b
         deploy:
           resources:
             reservations:
               devices:
                 - driver: nvidia
                   capabilities: [gpu]
       frontend:
         image: jasonc25/frontend:latest
         ports:
           - "8501:8501"
         environment:
           - API_URL=http://backend:8000/chat
           - SWITCH_MODEL_URL=http://backend:8000/switch_model
         depends_on:
           - backend
     ```

2. **设置模型路径**
   
   - 将模型放入本地目录（例如 `/path/to/models`）。
   
   - 设置环境变量：
   
     ```bash
     # Linux/Mac
     export PROFESSIONAL_MODEL_DIR=专业模型路径 
     export LIVING_MODEL_DIR=生活模型路径 
     # Windows
     set PROFESSIONAL_MODEL_DIR=专业模型路径   
     set LIVING_MODEL_DIR=生活模型路径

3.**运行**

```bash
docker-compose up
```
